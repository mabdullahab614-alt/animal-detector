# 🐾 Animal Detector & Cropper

An AI-powered web app that detects multiple animals in a photo, draws bounding boxes around each one, and crops them into individual images!

## 🌟 Live Demo
👉 [Try it here!](https://huggingface.co/spaces/BUDDDY2894830/animal-detector)

## 🎯 What It Does
- ✅ Upload any photo with animals
- ✅ Detects every animal automatically
- ✅ Draws colored boxes around each animal
- ✅ Crops each animal into individual pictures
- ✅ Shows confidence scores for each detection

## 🐾 Detectable Animals
🐱 Cat • 🐶 Dog • 🐦 Bird • 🐴 Horse • 🐑 Sheep
🐄 Cow • 🐘 Elephant • 🐻 Bear • 🦓 Zebra • 🦒 Giraffe

## 🛠️ Tech Stack
- **Python**
- **YOLOv8** — Object Detection
- **PyTorch** — Deep Learning
- **Gradio** — Web Interface
- **Hugging Face Spaces** — Deployment

## 🚀 How to Run Locally
```bash
pip install gradio torch torchvision ultralytics Pillow
python app.py
```
Then open: http://127.0.0.1:7860

## 📁 Files
- `app.py` — Main application
- `requirements.txt` — Dependencies
